## 学术研究 Agent

### 📖 学术论文智能问答系统
> 基于 RAG 技术的智能论文阅读助手，支持多轮对话和深度分析

**项目路径**: [`academic-paper-qa/`](./academic-paper-qa)

**核心特性**:
- 💬 **多轮对话**: 支持上下文记忆，连续追问
- 🧠 **智能问答**: 基于 RAG 的精准问答
- 📄 **多格式支持**: PDF、DOCX、Markdown、TXT
- 🔍 **语义检索**: 快速精准的向量检索
- 🌐 **联网搜索**: 集成 DuckDuckGo 获取最新信息
- 📊 **来源追溯**: 标注原文出处，可验证

**技术栈**: LlamaIndex, Chroma, Gradio, BAAI/bge-small-zh-v1.5

**适用场景**:
- 📖 快速理解学术论文
- 🔍 文献综述和对比分析
- 💡 研究灵感挖掘
- 🎓 学术教学辅导

[查看详情 →](./academic-paper-qa/README.md)

## 🎯 项目分类标准

### 按应用领域
- 🎓 **学术研究**: 论文分析、知识问答、文献综述
- 💼 **企业应用**: 客服、销售、运营自动化
- 👨‍💻 **开发工具**: 代码生成、调试、文档生成
- 🤝 **个人助理**: 日程管理、邮件处理、信息整理
- 🎨 **创意内容**: 文案创作、设计辅助、媒体制作
- 📊 **数据分析**: 数据处理、报告生成、可视化

### 按技术架构
- 🔍 **RAG (检索增强生成)**: 结合知识库的问答系统
- 🧠 **Multi-Agent**: 多智能体协作系统
- 🔧 **Tool-Using**: 具备工具调用能力的 Agent
- 💭 **Chain-of-Thought**: 复杂推理能力的 Agent
- 🔄 **ReAct**: 推理-行动循环框架

## 🛠️ 技术栈概览

### 主流框架
- **LlamaIndex**: RAG 应用开发框架
- **LangChain**: LLM 应用开发框架
- **AutoGPT**: 自主 Agent 框架
- **AgentGPT**: Web 界面的自主 Agent

### LLM 提供商
- **OpenAI**: GPT-3.5, GPT-4
- **Moonshot**: 中文优化模型
- **DeepSeek**: 高性价比选择
- **Anthropic**: Claude 系列
- **本地模型**: Llama, Mistral, Qwen

### 向量数据库
- **Chroma**: 轻量级向量存储
- **Pinecone**: 云端向量数据库
- **Weaviate**: 开源向量搜索引擎
- **Milvus**: 高性能向量数据库

## 🤝 如何贡献

我们欢迎各种形式的贡献！

### 添加新项目
1. Fork 本仓库
2. 在对应分类下创建项目目录
3. 添加完整的 README.md
4. 在主 README 中添加项目卡片
5. 提交 Pull Request

### 项目要求
- ✅ 完整的项目说明文档
- ✅ 清晰的安装和使用指南
- ✅ 可运行的示例代码
- ✅ 开源协议声明
- ✅ 技术栈和依赖说明

### 改进建议
- 报告 Bug 或问题
- 提出新功能建议
- 改进现有文档
- 分享使用经验
