# AI 模拟面试系统 - 功能介绍

## 🎯 系统简介

AI 模拟面试系统是一款基于大语言模型的智能面试辅助工具，旨在帮助求职者：
- 优化简历内容
- 准备面试问题
- 提升面试表现
- 增强求职竞争力

系统通过先进的自然语言处理技术，模拟真实的面试场景，为用户提供专业的反馈和建议。

## ✨ 核心功能

### 1. 📄 简历智能解析

**功能描述**：
- 支持 PDF 格式简历自动解析
- 自动提取简历关键信息
- 保留完整文本内容供后续分析

**技术特点**：
- 使用 PyMuPDF 高精度解析
- 自动处理多页简历
- 支持中英文混合内容

**使用场景**：
- 快速导入简历进行评估
- 为面试准备提供基础材料
- 批量处理多份简历

### 2. 🔍 多维度简历评估

**功能描述**：
系统从多个专业维度对简历进行全面评估，包括：

1. **基本信息完整性**（0-10分）
   - 联系方式是否完整
   - 个人信息是否清晰
   - 必要信息是否缺失

2. **工作经验相关性**（0-10分）
   - 工作经验与目标岗位的匹配度
   - 职业发展路径的连贯性
   - 工作年限的适配性

3. **项目经验质量**（0-10分）
   - 项目描述的详细程度
   - 技术难度和复杂度
   - 个人贡献的体现

4. **技能匹配度**（0-10分）
   - 技能栈与岗位要求的匹配
   - 技能深度和广度
   - 技能证明和应用实例

5. **教育背景**（0-10分）
   - 学历层次
   - 专业相关性
   - 学术成就

6. **整体印象**（0-10分）
   - 简历排版和格式
   - 语言表达质量
   - 专业度和可读性

**评估模式**：

- **完整评估**：提供详细的多维度分析报告
  - 各维度详细评分
  - 优点总结（3-5条）
  - 改进建议（3-5条）
  - 针对性优化方案

- **快速评分**：快速给出总体评分和简短评价
  - 总分（0-100分）
  - 一句话评价
  - 适合快速筛选

- **改进建议**：专注于提供可操作的优化建议
  - 内容完整性建议
  - 表达方式优化
  - 格式排版改进
  - 重点突出技巧
  - 针对性调整方案

### 3. 💼 智能模拟面试

**功能描述**：
基于上传的简历，AI 面试官会进行多轮对话式面试，模拟真实面试场景。

**面试类型**：

1. **技术面试**
   - 针对简历中的技术栈提问
   - 考察技术深度和广度
   - 算法和系统设计问题
   - 技术方案设计能力

2. **行为面试**
   - STAR 方法提问
   - 团队协作能力
   - 问题解决能力
   - 领导力和责任心

3. **综合面试**
   - 结合技术和行为
   - 职业规划探讨
   - 价值观匹配
   - 综合素质考察

**面试特点**：

- **针对性提问**：基于简历内容提出个性化问题
- **深度追问**：根据回答进行有针对性的深入探讨
- **场景模拟**：提出实际工作场景问题
- **专业反馈**：提供专业的面试表现反馈

### 4. 🌐 联网搜索增强

**功能描述**：
面试过程中，AI 可以联网搜索最新信息，验证候选人的回答，提供更真实的面试体验。

**应用场景**：

1. **技术验证**
   - 验证技术概念的准确性
   - 查询最新的技术趋势
   - 对比不同技术方案

2. **信息补充**
   - 获取行业最新动态
   - 查询公司背景信息
   - 补充相关知识点

3. **问题扩展**
   - 基于搜索结果提出新问题
   - 深入探讨热门话题
   - 考察学习能力

**技术实现**：
- 支持 DuckDuckGo 搜索引擎（免费、无需 API Key）
- 支持 SearXNG 公共实例
- 自动降级和容错机制

### 5. 🤖 多模型支持

**支持的 LLM 服务商**：

1. **OpenAI**
   - GPT-3.5-Turbo
   - GPT-4
   - GPT-4-Turbo

2. **DeepSeek**
   - DeepSeek-Chat
   - DeepSeek-Coder

3. **Qwen (通义千问)**
   - Qwen-Turbo
   - Qwen-Plus
   - Qwen-Max

4. **其他 OpenAI 兼容 API**
   - Moonshot
   - 智谱 AI
   - 本地部署模型

**灵活配置**：
- 简单的环境变量配置
- 支持自定义 API 地址
- 温度参数可调节

## 🎨 用户界面

### Web UI 设计

系统提供直观友好的 Web 界面，基于 Gradio 构建，包含四个主要标签页：

1. **📄 简历管理**
   - 文件上传组件
   - 状态显示
   - 内容预览

2. **🔍 简历评估**
   - 岗位信息输入
   - 评估模式选择
   - 结果展示

3. **💼 模拟面试**
   - 面试类型选择
   - 实时对话界面
   - 历史记录管理
   - 面试总结

4. **⚙️ 系统设置**
   - 系统信息查看
   - 配置状态显示

### 交互特点

- **响应式设计**：适配不同屏幕尺寸
- **实时反馈**：操作结果即时显示
- **友好提示**：详细的使用说明和提示
- **错误处理**：清晰的错误信息提示

## 🔒 隐私和安全

### 数据处理

- **本地处理**：简历文件在本地解析，不上传到第三方服务器
- **API 通信**：仅与配置的 LLM API 进行必要的通信
- **无数据存储**：不存储用户的简历内容到数据库

### 建议使用方式

- 在个人电脑上本地部署
- 使用可信的 LLM 服务商
- 定期清理日志文件
